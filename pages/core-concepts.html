What is Mod Reagan?

Mod Reagan is a tool, not a controller. It provides cross-server accountability by tracking user infractions across multiple servers. Each server operates independently, setting its own laws and moderation policies.

Hierarchical Roles

  • Citizens: General public, no moderation authority.
  • Enforcers: Moderators or server police.
  • Judges: Senior moderators handling disputes.
