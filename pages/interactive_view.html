Command Structure

The bot features tiered commands to ensure fair moderation. Commands are based on role hierarchy:

  • Citizens: Can request their own records, but only after moderator approval.
  • Enforcers: Record infractions, approve citizen requests, and submit evidence for review.
  • Judges: Review cases, resolve disputes, and apply final judgments.

Dashboard Overview

The dashboard for Mod Reagan provides an easy way to view and manage infractions:

  • Infractions by user
  • Pending case reviews
  • Cross-server infractions
  • Search and filter tools
